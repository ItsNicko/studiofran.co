About Me

17-year-old builder from Minot, ND. I make AI tools, mod circuits, and build whatever I'm curious about. Always learning by doing.

Philosophy

I build stuff I want to see exist. Sometimes it breaks. I fix it or rebuild it.

I don't have credentials, but I've got curiosity, time, and a little patience.

Everything here I've learned by opening things, reading docs, and messing up a lot.

Projects

  • ESP32 AI Assistant: AI dashboard with sensors, served over WiFi. Built with MicroPython + HA.
  • ChatGPT Clone: A working multi-turn chatbot using Flask + OpenAI API. Broke a lot. Learned a lot. (Discontinued)
  • Circuit-Bent Toy: Rewired a McDonald's toy with a voice FX chip. My first dive into hardware hacking.
  • Startup Deck: Built a fake startup brand: UI design, backend concept, and pitch plan.
  • HomeLab Server: Self-hosted PC for file storage, AI tools, and remote access using Docker + SSH.
  • Personal Website: This site. Hand-coded with just HTML and CSS. No frameworks.
  • IoT Weather Station (WIP): Planning a ESP32 weather station to send data to my HomeLab and control temperature.

Skills & Tools

  • Hardware: ESP32, Raspberry Pi, soldering, Orange Pi
  • Tools: Flask, Docker, Linux, Figma, TensorFlow, OpenCV
  • Systems: SSH, networking, REST APIs
  • AI: OpenAI API, BERT, ChatGLM
